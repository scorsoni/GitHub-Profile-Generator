# GitHub Profile Generator

A professional, modern, and interactive GitHub Profile README generator.

## Features
- **Interactive Editor**: Real-time preview of your README.
- **Customizable**: Add your bio, skills, socials, and projects.
- **Dynamic Stats**: Integration with `github-readme-stats` and `github-readme-streak-stats`.
- **Tech Stack**: Automatic badge generation for languages and tools.
- **Multi-language Support**: Generate profiles in English or Portuguese (BR).
- **Premium UI**: Glassmorphism design with dark mode.

## Tech Stack
- React
- Vite
- CSS Modules / Vanilla CSS (with variables)
- Lucide React (Icons)

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build**
    ```bash
    npm run build
    ```

## Project Structure
- `src/App.jsx`: Main application logic and state.
- `src/utils/generator.js`: Markdown generation logic.
- `src/components`: UI components.
- `src/index.css`: Global styles and themes.
